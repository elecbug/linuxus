# 🤝 Code of Conduct

## 📌 Our Principles

We aim to build a respectful and productive environment for everyone.

All contributors are expected to:

- Be respectful and constructive
- Focus on technical discussion
- Communicate clearly and concisely

---

## ✅ Expected Behavior

- Provide helpful and constructive feedback
- Accept criticism professionally
- Respect different levels of experience

---

## ❌ Unacceptable Behavior

- Personal attacks or insults
- Harassment or discrimination
- Dismissing others without explanation
- Repeated disruptive behavior

---

## ⚖️ Enforcement

Maintainers may:

- Request changes in communication style
- Close or reject PRs/issues if behavior is inappropriate
- Restrict participation if necessary

---

## 🧠 Philosophy

This is a **technical project**, not a social platform.

- We prioritize **clarity over emotion**
- We value **useful contributions over opinions**
